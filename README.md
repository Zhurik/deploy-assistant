# Deploy Assistant

Небольшая TUI тулзовина, чтобы автоматом деплоить кучу сервисов.

Если ваши выкатки в прод это создание тэга от main-а, то эта херня для вас.

## Запуск

Перед началом работы надо подготовить файл `services.yaml` со структурой:

```yaml
services:
  - name: <Имя сервиса для отображения в утилитке>
    local_path: <Абсолютный путь до репозитория>
```

```bash
uv run deploy-assistant
```

## Установка

Пока просто клонировать репу к себе и запускать из нее

## Разработка

Нужен Python 3.11+ и [uv](https://docs.astral.sh/uv/)
